Excel Calculation sync v_0.1

This little scrypt made for synchronization CB and feeding cable data from MDB file to DB files in the same directory.

Dependencies:
XLWings (to install type command: 'pip install xlwings')
PyQt5 (to install type command: 'pip install pyqt5')

Скрипт предназначен для синхронизации информации об отходящем автоматическом выключателе и питающем кабеле. В качестве исходной информации принимаются значения из расчетной формы
Главного распределительного щита (ГРЩ) или Вводно-рапределительного устройства (ВРУ). Далее эти данные прописываются в соответствующие ячейки в расчетной форме вторичного распределительного щита.

При запуске скрипта необходимо выбрать расчетный файл ГРЩ (файл эксель, начинающийся с "MDB_").  Скрипт берет названия отходящих линий и ищет такие же названия в файлах эксель (начинающихся с "DB_") в той же папке, где находится исходный файл MDB.

Запускаемый файл .exe не требует установки Python3 в систему.
